Attention , sous linux les scripts .sh sont
bien interprétés si les sauts de lignes sont encodés en LF
----------------
sous windows (avec notepad++ ou autre) , un fichier texte comporte des lignes
qui se terminent par CRLF
-----------------
notepad++ comporte le menu "affichage/symbole spéciaux/afficher tous les caractères"
pour visualiser les CRLF ou les LF
-----------------
notepad++ comporte le menu "edition / convertir les sauts de lignes /..."
pour convertir si besoin les sauts de lignes
